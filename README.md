# Notes Reading Trainer

https://pauldubois98.github.io/NotesReadingTrainer/

A web app to practise reading musical notes.
A note is displayed on a staff and you identify it by clicking a button or playing note on a piano.
Tracks response time, errors, and session stats.

## Run locally
### Dev
```bash
npm install
npm run dev
Go to [http://localhost:5173](http://localhost:5173)
```

### Build
```bash
npm run build      # Production build → dist/
npm run preview    # Preview the production build locally
npm run lint       # Lint with Biome
npm run format     # Auto-format with Biome
```
